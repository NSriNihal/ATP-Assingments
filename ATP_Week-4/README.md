# Week 4 — HTML5 Basics

## 📖 Topics Covered

- HTML5 Document Structure (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`)
- Semantic Elements (`<h1>`, `<h2>`, `<p>`, `<div>`)
- Lists (`<ul>`, `<li>`)
- Images (`<img>` with float styling)
- Links (`<a href>`)
- Tables (covered conceptually)
- Horizontal Rules (`<hr>`)

| File | Description |
