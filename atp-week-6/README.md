# Week 6 — MERN Stack Mini Projects

## 📖 Topics Covered

- Full-stack development with the MERN stack
- MongoDB with Mongoose (schemas, models, CRUD)
- Express.js backend API development
- React.js frontend with Vite
- Integration between frontend and backend
- Project organization and folder structure

## 📁 Project Structure

```
atp-week-6/
└── Mern-mini/
    ├── Employee-app/    # Employee management application
    ├── frontend/        # React frontend (Vite)
    ├── frontend-mern-mini/  # Additional frontend project
    └── Test/            # Testing folder
```

## 🛠️ Tech Stack

- **Frontend:** React (Vite)
- **Backend:** Node.js, Express.js
- **Database:** MongoDB with Mongoose
- **Tools:** Nodemon, REST Client

## ▶️ How to Run

```bash
# Backend
cd Mern-mini/Employee-app
npm install
node server.js

# Frontend
cd Mern-mini/frontend
npm install
npm run dev
```
