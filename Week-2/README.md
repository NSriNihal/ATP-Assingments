# Week 2 — Arrays, Objects, Classes & ES6 Modules

## 📖 Topics Covered

- Array Higher-Order Methods (`filter`, `map`, `reduce`, `find`, `findIndex`)
- Object Manipulation and Array of Objects
- ES6 Classes (constructor, methods, instances)
- Spread & Rest Operators
- Shallow Copy vs Deep Copy (`structuredClone`)
- ES6 Modules (`import` / `export`)
- Asynchronous JavaScript (`setTimeout`, `setInterval`)
