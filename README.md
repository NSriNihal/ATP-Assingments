# 📚 ATP Assignments

> A collection of weekly assignments completed as part of the **Advanced Training Program (ATP)** — covering JavaScript fundamentals through full-stack MERN development.

## 📑 Table of Contents

| Week | Topic | Folder |
|------|-------|--------|
| Week 1 | JavaScript Basics — Variables, Conditionals, Loops, Functions | [`Week-1`](./Week-1/) |
| Week 2 | Arrays, Objects, Classes, Spread/Rest, ES6 Modules, Async JS | [`Week-2`](./Week-2/) |
| Week 3 | Node.js & Express.js — REST API Development | [`ATP-Week-3`](./ATP-Week-3/) |
| Week 4 | HTML5 — Semantic Elements, Tables, Lists, Links | [`ATP_Week-4`](./ATP_Week-4/) |
| Week 5 | CSS3, Flexbox, Responsive Design & React (Vite) | [`ATP-Week-5`](./ATP-Week-5/) |
| Week 6 | MERN Stack Mini Projects | [`atp-week-6`](./atp-week-6/) |
| Capstone | Full-Stack Blog Application (MERN) | [`CapstoneProject`](./CapstoneProject/) |

## 🛠️ Tech Stack

- **Languages:** JavaScript (ES6+), HTML5, CSS3
- **Runtime:** Node.js
- **Backend:** Express.js, MongoDB, Mongoose
- **Frontend:** React (Vite), Vanilla CSS
- **Auth:** JWT, bcryptjs, cookie-based sessions
- **Cloud:** Cloudinary (image uploads)
- **Tools:** Nodemon, REST Client, Postman

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/NSriNihal/ATP-Assingments.git
cd ATP-Assingments

# Navigate to any week folder and follow its README for instructions
```
