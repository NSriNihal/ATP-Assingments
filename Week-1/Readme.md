# Week 1 — JavaScript Basics

## 📖 Topics Covered

- Variables (`let`, `const`)
- Conditional Statements (`if`, `else if`, `else`)
- Loops (`for`)
- Functions (declaration, parameters, return values)
- Arrays and Array methods (`splice`)
- Template Literals
