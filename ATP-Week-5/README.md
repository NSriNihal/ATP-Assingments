# Week 5 — CSS3, Flexbox & React

## 📖 Topics Covered

- CSS3 Fundamentals (selectors, box model, colors, typography)
- Flexbox Layout (`display: flex`, `justify-content`, `flex-direction`)
- Hover Effects & Box Shadows
- Responsive Design Patterns
- HTML Tables with `rowspan` / `colspan`
- React.js with Vite (component-based UI)

## 📁 Project Structure

```
ATP-Week-5/
├── index.html          # Modern landing page (hero + 3-column card layout)
├── style1.css          # CSS for the landing page (flexbox, hover effects)
├── 2/                  # Employee card layout (flexbox grid + images)
│   ├── index.html
│   └── style.css
├── 3/                  # HTML tables (rowspan, colspan, student marks)
│   ├── index.html
│   └── style.css
├── react-app-1/        # React app built with Vite
└── react-app-2/        # Second React app built with Vite
```

## 📝 Exercises

| Exercise | Description |
|----------|-------------|
| Landing Page (`index.html`) | Hero section with CTA button + 3-column content cards using Flexbox |
| Card Layout (`2/`) | Employee cards with images in a responsive flex grid |
| Tables (`3/`) | Student marks table with `rowspan` + expense table with `colspan` |
| React App 1 | Basic React component setup with Vite |
| React App 2 | Extended React app with routing/state |

## ▶️ How to Run

